Thinkpad Data:

This website was created to compare used laptop specifications against each other. Due to the large number of models and difficulties in cataloging each model, it is restricted to certain models from Lenovo for now. Each specification has been manually gathered from official specs sheets, which at some point will be downloadable from the website. Generally the data consists of the last 20 years of models from the X T and L series, with the E series on the way soon. In a perfect or at least better implementation, the data would be stored in a database or csv file and then actively processed by a backend program, likely python, then integrated back into the webpage by flask or something.
